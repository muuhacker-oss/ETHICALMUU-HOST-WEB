# 🌐 ETHICALMUU FREE HOSTING

**ETHICALMUU FREE HOSTING** ni mfumo rahisi wa PHP unaoruhusu watumiaji ku-host kurasa zao za web (**HTML, CSS, JavaScript**) kwa kutumia Custom URLs au kwa ku-upload faili la `.zip`.

## 🚀 Features
* **Direct Upload:** Upload faili moja la HTML, CSS, JS au faili la ZIP.
* **Code Editor:** Paste code za HTML moja kwa moja kwenye interface na uunde website yako.
* **Custom URLs:** Watumiaji wanaweza kuchagua jina la URL wanayotaka.
* **ZIP Automated Extraction:** Inafungua mafaili ya ZIP kiotomatiki kwenye server.
* **Storage & Stats Control:** Inaonyesha idadi ya sites zilizopo na jumla ya space iliyotumika.

```text

└── sites/             # Folder linalohifadhi web zilizouploadwa (Linajitengeneza lenyewe)
